# grafana
>http://grafana.org

 * 默认账号密码 admin admin
 * 这个有权限分层，字段自动发现，挺好

简单玩了一下，发现 grafana 很强大，基本需求完全可以满足。做一个简单介绍

 * 有账号体系，权限层级分明
 * 面板有4种： 图、表、值、文本
 * 自动索引检测
 * 有警戒值标记
 * 有模板、注解面板功能
 * 有面板分享，账号邀请功能
 * 混合datasource排版
