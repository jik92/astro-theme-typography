>虽然不喜欢微软这个公司，但是在做 VSCode 这个工具调研的时候发现功能已经整合 SublimeText Atom 的优点了

# 官网
http://code.visualstudio.com/
https://github.com/Microsoft/vscode

## 优点
 * 加载速度快, 自带 emmet
 * 插件不用翻墙，规范，整洁
 * Ctrl+P 兼容SublimeText 命令使用
 * 支持全局搜索， 正则搜索（高亮）
 * 支持 Git 文档管理，查看，diff
 * 支持 debug
 * 强大的 plugins （@popular）

## 结语
很适合给新手使用的 Editor，完整稳定的组件，开箱即用支持一整套开发流，不需要翻墙，掌握简单。
